## Introduction
Flight Fighter는 전투기를 조종해 1:1로 교전하는 아케이드 공중전 멀티플레이 게임입니다.
Photon PUN의 Room 기반 멀티플레이를 사용해, 플레이어는 룸(매치)에 입장하면 전투 씬으로 동기화되어 게임을 시작합니다.  

**Development**
- **기간**: 2022.11 ~ 2022.12
- **인원**: 1인 개발
- **환경**: Unity 2019.3.0f / Visual Studio 2021 / Windows 11
- **역할**: Gameplay(C#), 네트워크(매칭 , Photon PUN (클라이언트 기반)), UI·HUD, 사운드 연동

**Motivation & Intent**
-기말고사로 유니티로 3D 게임 제작 프로젝트가 있었습니다. 그 당시에 캡스톤 디자인으로 오픈월드 생존게임 싱글플레이를 제작했었어서  
멀티플레이 게임을 만들어 보자 하고 게임을 제작해보았습니다.
그리고 대부분 유튜브나 인터넷에 있는 강의작품을 토대로 작품을 만들어 (당시에는 GPT 같은 AI 대신 코드 짜주기가 없었습니다.) 인터넷에 본보기가
없는 그런 작품을 만들고 싶어 전투기 아케이드 게임을 만들게 되었습니다.
